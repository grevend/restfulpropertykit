The query post new value operator.

Makes a post request that will resolve to the new value or an error.

Usage:

let cancel = ($someQuery <- someNewValue).sink(...)

The query type Query must conform to the RestQuery protocol. Query.QueryParent and Query.QueryValue should equal Parent and Value.
