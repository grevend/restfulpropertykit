The query parameter concatenation operator.

Creates a copy of the on the left hand side provided query with an additional set of query parameters.

Usage:

let newQuery = $someQuery ++ ["someKey": someValue]

The query type Query must conform to the RestQuery protocol. Query.QueryParent and Query.QueryValue should equal Parent and Value. ParamKey and ParamValue must conform to both CustomStringConvertible and Hashable.
